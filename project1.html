Dark Dream [Photography shooting]

Service : Photography | Role : Photogrpaher

This project was for IAT233 spreadsheet design, we were assigned to create 3 spreadsheets with implementing the spatial design. The goal for this assignment was to train the ability to manage the space between elements, (elements such as images, text, headers, shapes or other design elements.) This project was individual and the photos were required to be taken by ourselves and had to be black and white. The entire process of the photoshoot was, First, picking the best photo shooting location, which was in downtown Vancouver, after that, I had to set the camera and set the correct aperture, IOS etc. After the photoshoot, there was a need for using Adobe photoshop to brush away useless background(people, lights, other objects, etc.) and used Adobe Lightroom to adjust the Saturation. The spreadsheets were done by using Adobe Illustrator.

This project is an individual project, the entire process was completed by myself. The challenges of this project were taking the photos which would match the content at the same time manage the design spacing with the content and image in the spreadsheet.
